# Real Data-Derived Trace Set

Source: [S-Labs/prompt-injection-dataset](https://huggingface.co/datasets/S-Labs/prompt-injection-dataset)

This repository includes 60 agent-style traces derived from real prompt-injection rows.
Each trace keeps source labels, prompt previews, policy-classifier events, synthetic latency fields,
and grader decisions so the viewer can be tested on realistic security-review traffic.
